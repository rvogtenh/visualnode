# visual-node-mvp

Audio-reactive WebGL shaders — controlled by microphone or line input.

---

## Quick Start

```bash
node server.js
# → http://localhost:3000
```

Open in Chrome, click **START**, allow microphone access.

No npm install needed — zero dependencies.

---

## Scenes

| Key | Scene |
|-----|-------|
| `1` | Interference Collapse |
| `2` | Stochastic Jump |
| `3` | Threshold Flip |
| `4` | Chromatic Storm |

---

## Controls

| Key | Action |
|-----|--------|
| `1`–`4` | Switch scene |
| `H` | Toggle info bar |
